# PlanX

**A comprehensive spatial planning plugin suite for QGIS**

PlanX is the core hub of a modular QGIS plugin ecosystem built for city planners and GIS professionals. It provides spatial statistics, zoning-rule workflows, urban-design analyses, and data utilities — all organized under a single PlanX menu with guided UI panels.

---

## Suite Overview

PlanX acts as the main plugin that ties together the following specialized sub-plugins:

| Plugin | Description |
|---|---|
| [PlanX CAD Toolset](https://github.com/YusufEminoglu/PlanX-CAD) | 18+ CAD drawing & editing tools with dockable panel |
| [PlanX UIP Toolset](https://github.com/YusufEminoglu/PlanX-UIP) | 8-stage automated Uygulama İmar Planı (Master Plan) workflow |
| [PlanX Settlement Toolset](https://github.com/YusufEminoglu/PlanX-Settlement) | 9-stage parametric settlement plan generation pipeline |
| [EasyFillet](https://github.com/YusufEminoglu/EasyFillet) | Standalone tangent arc fillet tool |

## Features

- Modular toolbox organized under a single PlanX processing menu
- Spatial statistics and zoning rule analysis
- Data import and export utilities
- Silent-mode batch processing capabilities
- Compatible with Turkish and international planning standards

## Installation

1. Download the latest `.zip` from [Releases](https://github.com/YusufEminoglu/PlanX/releases).
2. In QGIS: **Plugins → Manage and Install Plugins → Install from ZIP**.
3. Activate **PlanX** from the plugin list.
